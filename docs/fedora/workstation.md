## Initial Setup Time Zone Freeze Workaround
> [!NOTE]
> Seems to affect systems with certain NVIDIA cards.


Some may experience an issue after selecting a city from the search bar during Fedora Workstation initial setup after click next causing GNOME to freeze. To work around this click on a region on the map instead of using the search bar.

## Ulauncher Theme
 - [libadwaita-dark](https://github.com/kareemkasem/ulauncher-theme-libadwaita-dark) color theme

Installation
```console
mkdir -p ~/.config/ulauncher/user-themes

git clone https://github.com/kareemkasem/ulauncher-theme-libadwaita-dark \
  ~/.config/ulauncher/user-themes/libadwaita-dark
```
